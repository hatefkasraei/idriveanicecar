
# Web Development Starter Tool

### Features
- Automatic Build
- Browser-sync
- html partials ([Sniplicity](https://www.npmjs.com/package/sniplicity))
- Sass and Autoprefixer
- JS, Sass minify
- JS, Sass mapping

### Requirement
- npm

### Installation
- `$ npm install && npm run serve`

### Options
- `npm run serve:prod` for production
